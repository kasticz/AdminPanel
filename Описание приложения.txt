Суть приложения: пример простой административной панели, в которой можно работать с базами данных.

Использовано: 
HTML
CSS
PHP
MySQL


Реализовано:
1) В приложении представлены данные из 3 SQL баз данных: товары, пользователи и заказы.
2) С объектами баз данных можно взаимодействовать следующим образом: читать, создавать новые, редактировать существующие и удалять(CRUD).
3) В приложении слева на навигационной панели есть поиск, где можно искать товары по их названиям и id, а также пользоваетелей по их логинам.

Для разработки приложения использовался Open Server 5.4.3 с модулем PHP 8.1.
